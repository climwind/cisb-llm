# CISB Specification

## Vulnerability Description

**Source**
GCC Commit 02828845dd / Linux Kernel

**Description**
GCC -fdelete-null-pointer-checks optimization treats inline assembly memory operands as pointer dereferences, incorrectly eliminating subsequent null checks on those pointers.

**Evidence**
prefetch() macro contains asm with *(char *)(ptr). Compiler removes subsequent if (ptr) check. Runtime crash on NULL pointer.

**Requirement**
GCC compiler, optimization flags -O2, -O3, or -Os. Target architecture supporting the inline assembly.

**Mitigation**
Convert macros containing inline assembly with memory operands to static inline functions. Compile with -fno-delete-null-pointer-checks.
